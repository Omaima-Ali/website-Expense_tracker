Project Overview
This project aims to develop a simple and user-friendly Expense Tracker web application. Users can register, enter their monthly salary, create custom expense categories, add expenses, and track their remaining balance automatically.
________________________________________
Key Features
•	User authentication (registration and login)
•	Add and manage expense categories
•	Record expenses with title, category, amount, and date
•	Automatically update remaining salary after each expense
________________________________________
Database (MySQL) Implementation
The backend  use a MySQL database with the following:
•	Tables: users, categories, expenses, expense_logs
•	Basic CRUD operations: Insert, Update, Delete, Select for users, categories, and expenses
•	Triggers: Automatically log every insert and delete action on the expenses table
•	Stored Procedures: Add new expense with a balance check to prevent overspending
•	Stored Functions: Calculate remaining salary balance for a user
________________________________________
Technologies
•	MySQL for database management
•	Backend PHP and APIs to interact with the database
•	Frontend web interface (HTML, CSS)
